# 비밀번호 변경 페이지

### 기존 비밀번호

- O : 비밀번호 16자 이하 입력 제한
- O : 비밀번호 8자 미만시 에러메시지
- O : 소문자, 숫자, 특수문자 중 하나라도 미 포함시 에러메시지
- O : 허용된 특수문자 `~!@#$%^&\*()` 외 다른 특수문자 사용 시 에러메시지

### 변경 비밀번호

- O : 비밀번호 16자 이하 입력 제한
- O : 비밀번호 8자 미만시 에러메시지
- O : 소문자, 숫자, 특수문자 중 하나라도 미 포함시 에러메시지
- O : 허용된 특수문자 `~!@#$%^&\*()` 외 다른 특수문자 사용 시 에러메시지

### 변경 비밀번호 확인

- O : 입력 16자 제한
- O : 입력 비밀번호와 다를 경우 에러메시지

### 비밀번호 변경

- X : 기존 비밀번호 맞는지 확인 요청
- X : 모든 조건 충족 시 비밀번호 변경 요청
